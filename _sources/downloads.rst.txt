It you use Neper for your own work, please mention it explicitly and cite one or several of the papers given in the References section below.  The papers are available for download.

Current stable release
----------------------

- `neper-3.4.0.tgz (15 Apr 2019) <http://sourceforge.net/projects/neper/files/neper-3.4.0.tgz/download>`_: source code and documentation
- `neper.pdf <docs/neper.pdf>`_: documentation

Development repository
----------------------

A development repository is hosted on GitHub:
http://github.com/rquey/neper.  It provides the development version.
Code contributions to be included in the official (public) version of
Neper should be submitted as pull requests.

Supplementary software
----------------------

- Dependencies:
  `GSL  <http://www.gnu.org/software/gsl/>`_,
  `muParser  <http://beltoforion.de/article.php?a=muparser>`_ (included in Neper),
  `nanoflann  <https://github.com/jlblancoc/nanoflann>`_ (included in Neper),
  `Gmsh <http://geuz.org/gmsh>`_,
  `libScotch <http://www.labri.fr/perso/pelegrin/scotch>`_,
  `POV-Ray <http://www.povray.org>`_
- All versions of Neper can be accessed `here <http://sourceforge.net/projects/neper/files/neper/>`_.

Mailing lists
-------------

There are two mailing lists for Neper, which you may find useful to
subscribe to:

- `neper-announce <https://lists.sourceforge.net/lists/listinfo/neper-announce>`_: the "read-only" list for important news: new releases, bug fixes, etc. (low traffic, highly recommended!).
- `neper-users <https://lists.sourceforge.net/lists/listinfo/neper-users>`_: the "read-write" list for users. Please send all questions, bug reports, requests or any errors or omissions in this manual to this list.
